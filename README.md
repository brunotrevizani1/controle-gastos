💰 Controle de Saldo

Sistema web simples e intuitivo para controle de gastos pessoais, permitindo acompanhar saldo, despesas no cartão e lembretes do dia a dia.

Desenvolvido com HTML, CSS e JavaScript puro, com foco em uso pessoal.

🚀 Funcionalidades

💵 Salário / Saldo
Definição do valor mensal
Edição direta clicando no valor
Cálculo automático do saldo restante

💳 Gastos no Cartão

Adicionar despesas
Informar nome e valor
Remover itens facilmente
Atualização automática do total gasto

🧮 Calculadora integrada

Operações básicas:
Soma
Subtração
Multiplicação
Divisão
Botão para limpar valores

📝 Lembretes

Criar anotações rápidas
Estilo “post-it”
Organização visual simples

📱 Responsividade

Funciona no celular e no computador
Layout adaptado para telas menores

💾 Armazenamento

O sistema utiliza:
localStorage do navegador

✔️ Vantagens:
Dados salvos automaticamente
Não precisa de login
Funciona offline
⚠️ Limitações:
Dados ficam apenas no dispositivo atual
Limpar cache ou usar outro navegador apaga os dados

🛠️ Tecnologias utilizadas

HTML5
CSS3
JavaScript (puro)

🌐 Acesso

O sistema pode ser acessado diretamente pelo navegador:
